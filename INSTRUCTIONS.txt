🚀 PROCHAINES ÉTAPES (DANS L'ORDRE)

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 1 : INSTALLER NODE.JS                                 │
└──────────────────────────────────────────────────────────────┘

📌 MÉTHODE AUTOMATIQUE (recommandee) :

1. Double-cliquez sur: install-nodejs.bat
2. Attendez la fin du telechargement et de l'extraction
3. Fermez TOUS les terminaux ouverts
4. Ouvrez un NOUVEAU terminal PowerShell
5. Verifiez l'installation:
   
   node -v
   npm -v

   ✅ Si vous voyez les numeros de version, c'est bon !

📌 MÉTHODE MANUELLE (si le script echoue) :

1. Allez sur: https://nodejs.org/
2. Telechargez la version LTS (Long Term Support)
3. Lancez l'installateur node-vXX.XX.X-x64.msi
4. Suivez les etapes (Accepter, Next, Next, Install)
5. Redemarrez votre ordinateur
6. Verifiez: node -v et npm -v

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 2 : INSTALLER LES DÉPENDANCES DU PROJET               │
└──────────────────────────────────────────────────────────────┘

Une fois Node.js installe:

1. Ouvrez PowerShell
2. Naviguez vers le projet:
   
   cd C:\Users\PC\portfolio

3. Installez les dependances (Astro + Vercel adapter):
   
   npm install

   ⏱️ Durée: ~30 secondes à 1 minute

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 3 : TESTER LOCALEMENT                                 │
└──────────────────────────────────────────────────────────────┘

Lancez le serveur de developpement:

npm run dev

✅ Ouvrez votre navigateur: http://localhost:4321

Vous devriez voir votre portfolio en direct !

🔄 Le serveur recharge automatiquement à chaque modification.
🛑 Pour arreter: Ctrl+C dans le terminal

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 4 : PERSONNALISER LE CONTENU                          │
└──────────────────────────────────────────────────────────────┘

📝 Editez UN SEUL fichier: src\data\profile.ts

Vous pouvez modifier:

✏️ Informations personnelles (nom, titre, avatar)
✏️ Statistiques affiches dans le Hero
✏️ Section About (presentation personnelle)
✏️ Competences techniques et soft skills
✏️ Parcours academique (education)
✏️ Experiences professionnelles
✏️ Projets personnels/academiques
✏️ Certifications
✏️ Publications (si applicable)
✏️ Liens sociaux (GitHub, LinkedIn, etc.)

💡 Les sections VIDES sont automatiquement MASQUÉES.
   Remplissez uniquement ce qui vous concerne !

Exemple pour ajouter une education:

]

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 5 : POUSSER VERS GITHUB                               │
└──────────────────────────────────────────────────────────────┘

📌 MÉTHODE 1 : Via GitHub CLI (recommande)

1. Installez GitHub CLI: https://cli.github.com/
2. Authentifiez-vous:
   
   gh auth login
   
   → Choisissez: GitHub.com
   → Choisissez: HTTPS
   → Authentifiez-vous via le navigateur web
   
3. Configurez Git pour utiliser GitHub CLI:
   
   gh auth setup-git

4. Poussez le code:
   
   cd C:\Users\PC\portfolio
   git push -u origin main

📌 MÉTHODE 2 : Via interface GitHub (alternative)

Si GitHub CLI ne fonctionne pas:

1. Allez sur: https://github.com/Med-Assam/portfolio
2. Le depot existe deja (vide pour l'instant)
3. Cliquez sur "uploading an existing file"
4. Glissez-deposez TOUS les fichiers du dossier
   C:\Users\PC\portfolio
   (sauf .git, node_modules si presents)
5. Commit avec message: "Initial portfolio setup"

┌──────────────────────────────────────────────────────────────┐
│ ÉTAPE 6 : DÉPLOYER SUR VERCEL                               │
└──────────────────────────────────────────────────────────────┘

📌 MÉTHODE 1 : Dashboard Vercel (LA PLUS SIMPLE)

1. Allez sur: https://vercel.com
2. Cliquez sur "Sign Up" ou "Log In"
3. Connectez-vous avec votre compte GitHub
4. Cliquez sur "New Project"
5. Importez le repo: Med-Assam/portfolio
6. Vercel detecte automatiquement Astro
7. Cliquez sur "Deploy" (ne changez RIEN)
8. Attendez ~1 minute (build + deploiement)
9. ✅ Votre site est en ligne !

10. Pour l'alias personnalise:
    → Allez dans Settings > Domains
    → Add Domain: portfolio-mohamed-assam.vercel.app
    → Vercel configure automatiquement l'alias

📌 MÉTHODE 2 : Vercel CLI (avance)

Si vous preferez la ligne de commande:

1. Installez Vercel CLI globalement:
   
   npm install -g vercel

2. Authentifiez-vous:
   
   vercel login
   
   → Ouvrez le lien dans le navigateur
   → Confirmez la connexion

3. Deployez:
   
   cd C:\Users\PC\portfolio
   vercel --prod

4. Configurez l'alias:
   
   vercel alias set <url-temporaire> portfolio-mohamed-assam.vercel.app
